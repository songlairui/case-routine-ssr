# 创建 vue 服务端渲染脚手架

## 运行方式

```
npm run build  // 编译
```

```
npm start    // 启动服务
```
访问 http://localhost:8003 为主页

点击链接跳转到 http://localhost:8003/about
在此页面刷新，得到的html源代码是刷新过了的。
